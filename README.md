# The Last Train Home

An interactive web experience — a quiet journey through five hand-crafted scenes.

## The Journey

You arrive at a countryside train station at golden hour.  
You walk through open fields.  
You pass through a forest shrine.  
You rest by a moonlit lake.  
You return as the last train arrives.

Four memories to find along the way.

## Built With

- JavaScript
- GSAP (animations)
- Web Audio API (ambient sound)
- Pure CSS 

## How To Experience

Open the link, turn your sound on, and take your time.  
This is not a game. There is no score.  
Just a feeling.
